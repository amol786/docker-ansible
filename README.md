# Docker-Ansible
<p>
Docker image for Ansible software, installed in a Ubuntu OS to 
execute ansible playbooks in stable version for Network CLI
Configurations
</p>

Base OS
Ubuntu 16.04

Python version 
3.6

Ansible Version 
2.9
